# Decisions Log

A running record of significant decisions made on this project, so future work (by a person or an AI agent) doesn't re-litigate settled questions or lose the reasoning behind them.

Add a new entry every time you make a decision that changes direction, kills an idea, commits to an approach, or resolves an open question.

## Template for each entry

### [YYYY-MM-DD] Decision title

**Context:** What question or fork in the road prompted this decision?

**Evidence:** What real evidence (competitor research, user feedback, test results, market data) informed it? Link to sources where possible.

**Decision:** What was decided, stated plainly.

**Why:** Why this option over the alternatives.

**Revisit when:** What would have to change for this decision to be worth reopening (a date, a metric, new evidence)? If nothing, write "Not planned."
